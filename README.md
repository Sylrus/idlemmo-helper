# IdleMMO Helper

</p>

Extension Chrome d'amélioration de la QOL pour [IdleMMO](https://www.idle-mmo.com/)

## Fonctionnalités

- ⚔️ **Calculateur Battle XP** sur la page de combat (scale, hits, bonus, potions, Energizing Pool).
- 📊 **Bandeau de suivi** en haut de l'écran : statut Assaut, World Boss, Energizing Pool ("Pool UP").
- 💰 **Prix net après taxe** affiché automatiquement sur le marché.
- 🛒 **Vendor Check** dans l'inventaire pour comparer prix vendeur vs prix net du marché.
- Et bien + dans les prochaines mises à jour

## Installation

### Installation manuelle (mode développeur)

1. Télécharge ou clone ce dépôt :
   ```bash
   git clone https://github.com/Sylrus/idlemmo-helper.git
   ```
2. Ouvre Chrome et va sur `chrome://extensions`.
3. Active le **"Mode développeur"** (en haut à droite).
4. Clique sur **"Charger l'extension non empaquetée"** (Load unpacked).
5. Sélectionne le dossier `idlemmo-helper` que tu viens de télécharger/cloner.
6. L'extension apparaît dans la liste et est immédiatement active sur idle-mmo.com.

> Pour mettre à jour en mode manuel : fais un `git pull` (ou re-télécharge le zip), puis clique sur le bouton "Recharger" de l'extension dans `chrome://extensions`.
